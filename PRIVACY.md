# 隐私政策 / Privacy Policy

**最后更新 / Last Updated: 2026-05-16**

## 中文

### 概述

iMonet 是一款本地图片查看工具。我们非常重视您的隐私。本隐私政策说明了 Monet 如何处理您的数据。

### 数据收集

**iMonet 不收集任何个人数据。** 具体来说：

- **不上传任何内容**：iMonet 完全在本地运行。您查看的所有图片、文件和文件夹数据均仅存储在您的设备上。
- **无分析统计**：iMonet 不包含任何分析 SDK、统计追踪或崩溃报告工具。
- **无账户系统**：iMonet 无需注册、登录或创建账户。
- **无第三方数据共享**：iMonet 不会向任何第三方传输数据。

### 数据存储

- **书签数据**：iMonet 使用系统提供的安全书签（Bookmark Data）来记住您授权过的文件夹访问权限。这些数据仅存储在您设备的 `UserDefaults` 中，不会上传至任何服务器。
- **用户偏好**：您的设置偏好（如自动索引开关）同样存储在 `UserDefaults` 中，完全本地保存。

### 网络访问

Monet 本身不发起任何网络请求。唯一可能的外部链接是您主动点击"贡献"或"报告问题"按钮时，系统会调用系统浏览器打开 GitHub 页面。这一行为完全由您主动触发。

### 第三方服务

iMonet 不集成任何第三方数据分析、广告或追踪服务。

### 变更

如本隐私政策发生变更，我们会在本页面更新内容。

### 联系我们

如有任何隐私相关问题，请通过 GitHub Issues 联系我们：https://github.com/wflixu/Monet/issues

## English

### Overview

iMonet is a local image viewer application. We take your privacy seriously. This policy explains how Monet handles your data.

### Data Collection

**iMonet does not collect any personal data.** Specifically:

- **No content upload**: iMonet runs entirely locally. All images, files, and folder data remain on your device.
- **No analytics**: iMonet contains no analytics SDK, usage tracking, or crash reporting tools.
- **No user accounts**: iMonet does not require registration, login, or account creation.
- **No third-party data sharing**: iMonet does not transmit any data to third parties.

### Data Storage

- **Bookmark Data**: iMonet uses system-provided security-scoped bookmarks to remember your authorized folder access permissions. This data is stored locally in `UserDefaults` on your device and is never uploaded to any server.
- **User Preferences**: Your settings (such as auto-indexing toggle) are also stored in `UserDefaults` locally.

### Network Access

iMonet itself does not initiate any network requests. The only possible external link is when you actively click the "Contribute" or "Report a Bug" button, which opens a GitHub page in your system browser. This action is entirely user-initiated.

### Third-Party Services

iMonet does not integrate any third-party analytics, advertising, or tracking services.

### Changes

If this privacy policy changes, we will update the content on this page.

### Contact

If you have any privacy-related questions, please reach out via GitHub Issues: https://github.com/wflixu/Monet/issues
